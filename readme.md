## Install

```html

```
##### x-draggable
```html
<script src="https://cdn.jsdelivr.net/gh/PowerPeps/Alpenstock@master/src/x-draggable.js"></script>
```

```html
<script src="https://cdn.jsdelivr.net/gh/PowerPeps/Alpenstock@master/src/x-icon.js"></script>
```
