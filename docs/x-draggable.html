x-draggable

Makes an element draggable within one or more $refs containers.

Usage

<div x-data>
  <div x-ref="zone">
    <div x-draggable="zone">Drag me</div>
  </div>
</div>

Example

Drag me

Expression

Space-separated names of $refs containers. The element is constrained within the union of all containers. When dragged into another container, it is reparented to it.

Modifiers

.opacity reduces opacity when dragged outside any container

Notes

Supports mouse and touch events. The element receives position: absolute automatically.
