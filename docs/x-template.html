x-template

Renders {expression} placeholders in element content using Alpine reactive data.

Usage

<div x-data="{ name: 'world', count: 0 }">
  <p x-template>Hello, {name}! Clicked {count} times.</p>
  <input x-model="name" placeholder="name">
  <button @click="count++">Click</button>
</div>

Example

Hello, {name}! Clicked {count} times.

Expression

None. The directive reads the element's innerHTML as a template.

Syntax

{key} evaluated as an Alpine expression
\{ \} renders as literal braces

Modifiers

.html renders result as innerHTML instead of textContent

Notes

The template is captured once at init. Placeholders are reactive via Alpine.effect.
