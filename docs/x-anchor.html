x-anchor

Anchors an element to a $refs target using the CSS Anchor Positioning API.

Usage

<div x-data>
  <button x-ref="btn">Anchor target</button>
  <div x-anchor.bottom.right="btn">Anchored element</div>
</div>

Example

Anchored element

Requires a Chromium-based browser (CSS Anchor Positioning API).

Expression

Name of a $refs reference to anchor to.

Modifiers

.top aligns to the top edge of the anchor
.bottom aligns to the bottom edge
.left aligns to the left edge
.right aligns to the right edge

Modifiers can be combined: .bottom.left

Notes

Requires the CSS Anchor Positioning API. Currently supported in Chromium-based browsers.
