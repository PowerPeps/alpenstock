x-icon

Injects an SVG icon from a provider into the element.

Usage

<span x-icon:pixel="arrow-left" class="w-5 h-5"></span>
<span x-icon:lucide="circle-check" class="w-5 h-5"></span>

Example

arrow-left
arrow-right
github
copy
check
drag-and-drop
lucide: star
lucide: zap

SVG attributes (class, style, etc.) are forwarded to the injected icon.

Expression

Name of the icon within the provider.

Providers

:pixel pixelarticons — pixel art icon set
:lucide lucide — clean, consistent icon set

Notes

SVG responses are cached per URL. The element is replaced by the injected <svg> node. All non-x-icon attributes are forwarded.
