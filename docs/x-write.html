x-write

Reveals text character by character.

Usage

<p x-write="40">The quick brown fox jumps over the lazy dog.</p>

Example

Expression

Delay between characters in milliseconds. Defaults to 0.

Modifiers

.[ms] start delay in milliseconds before the animation begins 
<p x-write.500="30">Starts after 500ms</p>

Notes

Sets aria-label to the original text for accessibility. The animation uses requestAnimationFrame.
