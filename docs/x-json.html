x-json

Fetches JSON from a URL and renders it via a jsonata expression.

Usage

<span x-json="https://jsonplaceholder.typicode.com/todos/1 | title"></span>

Example

title:

Fetched live from jsonplaceholder.typicode.com.

Expression

"url | jsonata expression" — omit the expression to render the full JSON response.

Modifiers

.html renders as innerHTML instead of textContent

Dependencies

Requires jsonata loaded before Alpine.

Notes

Responses are cached per URL. The request is aborted on cleanup.
